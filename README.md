# SearchApp
SwiftUI / 네이버 검색 API를 활용한 검색 앱

## 사용 API
- [네이버 검색 API](https://developers.naver.com/docs/serviceapi/search/blog/blog.md#%EB%B8%94%EB%A1%9C%EA%B7%B8)

## 스펙
- 검색 메인 화면에서 검색어를 입력
- TabView를 통해 블로그, 뉴스, 이미지 검색 결과 반환
- 결과 목록을 눌렀을 때 WebView로 이동
- 검색 결과에서 검색어 하이라이트
